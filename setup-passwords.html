Setup User Passwords

This utility sets passwords for all Indonesia Power team members using database authentication

Instructions:

  1. Set the default password (or use the default: demo123)
  2. Click "Hash Passwords for All Users" to set passwords for all team members
  3. Passwords are securely hashed using bcrypt in the database
  4. Users can change their password after first login
